# Claude Chatbot


## About
An Ai chatbot  project that can answer various questions, it has tons of knowledge in its database to have the answer to your question

## Live demo Link
[Click here to try the chatbot](https://glenchatbot.streamlit.app/)

## Tech stack
Python, Anthropic Claude API, Streamlit

## How to run locally 
1. Clone this repo
2. Install dependencies: pip install -r requirements.txt
3. Add your API key to a .env file: ANTHROPIC_API_KEY=your-key
4. Run: python -m streamlit run app.py
